# 패션 트렌드 수명주기 분류

> Google Trends 검색 데이터를 **STL 시계열 분해**로 분석해, 패션 아이템을
> **메가 · 마이크로 · 스테디** 트렌드로 분류하고 **수명주기 단계와 실무 전략**까지 도출한 데이터 분석 프로젝트.

### 🔗 [라이브 데모 보기 →](https://yeonjung77.github.io/stl/)

> 분석 결과 시각화 + **키워드를 입력하면 즉석 분류해 주는 실시간 분류기**가 한 페이지에 담겨 있습니다.

<br>

## 한눈에

|  |  |
|---|---|
| **데이터** | Google Trends 검색 관심도 (한국 · 최근 5년 · 주별) |
| **규모** | 패션 아이템 **156개** |
| **핵심 기법** | STL 시계열 분해 · K-means 군집화 · 규칙기반 수명주기 판정 |
| **산출물** | 인터랙티브 웹페이지 + 실시간 분류 앱 |

<br>

## 분석 흐름

```
검색 곡선 ──STL분해──▶ 트렌드·계절성·잔차 ──▶ 7개 지표 ──▶ 군집화 ──▶ 분류 + 수명주기
```

1. **STL 분해** — 검색 곡선을 `트렌드 + 계절성 + 잔차`로 분리
2. **지표 산출** — 분해 성분에서 정량 지표 7종 계산 (트렌드강도·계절성·기울기·지속기간·잔차변동·첨예도·피크시점)
3. **군집화** — K-means(k=3)로 메가/마이크로/스테디 분류 — *여러 모델·k를 실루엣 점수로 비교해 타당성 검증 ([보고서](stl200/MODEL_COMPARISON.md))*
4. **수명주기 판정** — 트렌드 방향 + 현재/피크 비율로 도입~쇠퇴 단계 추정

<br>

## 핵심 포인트

- **정답 없는 분류를 정직하게** — 지도학습 대신 군집화 + 규칙으로 사후 검증
- **모델을 객관적으로 검증** — 실루엣 점수로 알고리즘·군집 수 비교
- **분석을 제품으로** — 학습된 모델을 저장해, 새 키워드를 실시간 분류하는 앱으로 배포

<br>
